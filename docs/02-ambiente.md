# Etapa 1 – Ambiente do Laboratório

## Host
- Sistema Operacional: Windows
- Memória RAM: 8 GB
- Disco disponível: 60 GB

## Virtualização
- Ferramenta: VMware Workstation Player
- Tipo de virtualização: Desktop (local)

## Objetivo do Ambiente
Simular um servidor Linux semelhante a ambientes reais de redes e infraestrutura, permitindo a prática de configuração e diagnóstico via terminal.
