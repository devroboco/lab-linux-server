# Etapa 5 – Preparação do Servidor Linux

## Objetivo
Preparar o servidor Linux para operação contínua e instalação de serviços, aplicando atualizações, ferramentas essenciais e boas práticas iniciais.

## Atualização do Sistema
Foram executados os comandos:
- sudo apt update
- sudo apt upgrade

Com isso, o sistema foi atualizado com os pacotes mais recentes disponíveis.

## Identificação do Sistema
A versão do sistema operacional foi verificada para garantir compatibilidade com os serviços a serem instalados.

## Definição do Hostname
O hostname do servidor foi configurado para facilitar a identificação na rede e em sessões administrativas remotas.

## Ferramentas Instaladas
Foram instaladas ferramentas essenciais para diagnóstico e administração de redes, incluindo ping, traceroute, ferramentas DNS e utilitários de monitoramento.

## Conclusão
Ao final desta etapa, o servidor encontra-se atualizado, identificado corretamente e preparado para receber serviços de rede.
