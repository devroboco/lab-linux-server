# Etapa 2 – Máquina Virtual Linux

## Configuração da VM
- Memória RAM: 2 GB
- Processadores: 2 vCPUs
- Disco: Virtual (VMDK)
- Rede: NAT

## Sistema Operacional
- Linux Server (imagem pré-configurada)
- Interface: terminal (sem ambiente gráfico)

## Justificativa
A utilização de Linux sem interface gráfica reflete a realidade de servidores em ambientes corporativos, proporcionando melhor desempenho e maior segurança.
